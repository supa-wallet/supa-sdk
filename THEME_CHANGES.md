### 2. Переключение светлой/темной темы в Demo

Добавлена полная поддержка переключения между светлой и темной темой:

#### Новые файлы:
- **`demo/src/ui/ThemeProvider.tsx`** - React контекст-провайдер для управления темой
- **`demo/src/ui/theme.ts`** - Добавлены две темы (`darkTheme` и `lightTheme`)
- **`demo/src/types/styled.d.ts`** - TypeScript декларация для styled-components

#### Изменения в существующих файлах:

**`demo/src/ui/theme.ts`:**
- Разделена одна тема на `darkTheme` (темная) и `lightTheme` (светлая)
- Темная тема осталась такой же, как была
- Добавлена светлая тема с адаптированными цветами

**`demo/src/ui/GlobalStyles.tsx`:**
- Обновлены стили для использования темы из контекста (через `props.theme`)
- Добавлен плавный transition для изменения цветов

**`demo/src/ui/styled.ts`:**
- Все styled-компоненты обновлены для использования `theme` из пропсов
- Добавлена поддержка `$fullWidth` в компонент `Flex`

**`demo/src/App.tsx`:**
- Добавлен импорт `ThemeProvider` и `useTheme`
- Обернут `SupaProvider` в `ThemeProvider`
- Добавлена кнопка переключения темы в header (иконки Sun/Moon)
- Добавлена кнопка переключения темы на экране логина
- Все styled-компоненты обновлены для использования темы из контекста

**`demo/src/ui/index.ts`:**
- Экспортируется `ThemeProvider`, `useTheme`, `darkTheme`, `lightTheme`

#### Функционал:

1. **Автоматическое определение темы** - При первом запуске используется системная тема (dark/light)
2. **Сохранение в localStorage** - Выбранная тема сохраняется между сессиями
3. **Переключатель в UI** - Кнопки с иконками солнца/луны в header и на экране логина
4. **Плавные переходы** - Анимация при изменении цветов (250ms)

#### Использование:

```tsx
import { ThemeProvider, useTheme } from './ui';

// В корневом компоненте
<ThemeProvider>
  <App />
</ThemeProvider>

// В любом дочернем компоненте
const { mode, toggleTheme, setTheme } = useTheme();

// Переключить тему
toggleTheme();

// Или установить конкретную
setTheme('dark');
setTheme('light');
```
